# Юрий Берингов - Медитации и Саунд-Дизайн

Создание профессиональных медитаций и wellness-аудио с использованием AI Suno.

## ⚡ Быстрый старт

Установите зависимости и запустите локально:

```bash
npm install
npm run dev
```

Сайт будет доступен по адресу: http://localhost:4321

## 📋 Содержимое

- **Главная** - Призыв к действию и обзор услуг
- **Услуги** - Детальное описание коммерческих предложений
- **Портфолио** - Примеры работ (медитации для йоги, сна, AI-generated)
- **О себе** - История перехода к медитациям
- **Контакты** - Форма заказа

## 🏗 Технологический стек

- **Astro 4.x** - Статический генератор сайтов
- **Tailwind CSS** - Стилизация
- **GitHub Actions** - Автоматический деплой
- **GitHub Pages** - Хостинг

## 🚀 Деплой

Настройте GitHub Actions workflow (файл в `.github/workflows/astro.yml`) для автоматического деплоя на GitHub Pages:

1. Создайте репозиторий на GitHub
2. Запушьте:
   ```bash
   git remote add origin https://github.com/BERINGOV/beringov-pro.git
   git push -u origin main
   ```
3. В настройках репозитория включите GitHub Pages (Settings → Pages → Branch: main)

## 📝 Контент

Сайт изначально имеет placeholder-контент. Для замены:
- Редактируйте Astro компоненты в `src/pages/`
- Добавляйте аудио в `public/audio/`
- Обновляйте изображения в `public/images/`
- Редактируйте тексты в `src/styles/main.css` для кастомизации цветов

## 🎨 Дизайн

Цветовая палитра "warm spa" - спокойные, заземляющие тона:
- #F5F0E8 (warm cream) - фон
- #87A87A (sage green) - акценты
- #2C3E50 (deep blue-grey) - текст
